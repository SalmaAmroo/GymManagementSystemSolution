# Gym Management System (ASP.NET MVC)

A simple ASP.NET MVC web application for managing gym operations such as members, trainers, classes, and attendance.  
This project was created for learning and practice purposes to demonstrate the MVC architecture.

---

## Features
- Member management (Create, Read, Update, Delete)
- Trainer management
- Class scheduling
- Attendance tracking
- Authentication and basic roles (Admin / User)

---

## Tech Stack
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML, CSS, Bootstrap

---

## How to Run the Project
1. Open the solution file in **Visual Studio**
2. Update the connection string in `appsettings.json` (or `web.config`)
3. Build the solution
4. Run the project using **IIS Express** or **Kestrel**

---

## Project Structure
- **Models**: Application data models  
- **Views**: Razor views (UI)  
- **Controllers**: Handle requests and business logic
